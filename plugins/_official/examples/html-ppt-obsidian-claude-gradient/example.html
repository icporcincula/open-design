04 / 08
● WORKFLOW · 4 STEPS

从想法到交付，一条工作流

1

选择起点

一句话描述目标，或从模板 / Plugin 直接选起点。

2

确定视觉方向

选定方向后，色板、字体、间距自动带入生成流程。

3

生成 Artifact

Agent 读取全部上下文，产出真实可运行的文件，沙盒内即时预览和修改。

4

交付或制作视频，记忆沉淀

导出给工程继续开发，或用 HyperFrames 直接转为营销视频；每次选择都成为下次生成的上下文。

● LOCAL DATA

你的数据，只在你的机器上

// .od/ —— Open Design 的全部运行数据都在这个本地目录
.od/
├── app.sqlite          // 本地 SQLite，会话与设计记忆
├── projects/           // 每个项目的 Agent 工作目录
├── artifacts/          // 生成的原型 / Slides / 视频产物
└── media-config.json   // BYOK 凭据，API Key 不出机器

无需注册、没有云端同步 —— 可 git、可 diff、可回滚，设计产物终于能版本控制。
